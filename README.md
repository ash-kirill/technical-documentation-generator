# technical-documentation-generator

# 📄 Автоматическая генерация технической документации (АОСР)

## 📌 Что это?
Go-приложение для автоматической генерации технической документации АОСР (акт освидетельствования скрытых работ) с распознаванием штампов на чертежах через Yandex Vision API (OCR).

**Результат:** обработка документа — с 30 минут до 20 секунд.

---

## 🎯 Проблема
Инженеры ПТО тратят часы на ручное заполнение АОСР. Это приводит к:
- Ошибкам в документации
- Потере времени
- Задержкам в согласовании

---

## ✅ Решение
Приложение:
1. Загружает скан чертежа
2. Распознаёт штамп через Yandex Vision API (OCR)
3. Извлекает данные: номер чертежа, дату, этаж и так далее...
4. Генерирует готовый документ

---

## 📊 Результаты
- Сократил обработку с 30 минут до 20 секунд
- Исключил ошибки ручного ввода
- Повысил скорость согласования

---

## 🛠 Технологии
- **Go** — основной язык
- **Yandex Vision API** — OCR-распознавание
- **REST API / JSON** — работа с API
- **PostgreSQL** — в планах (для аналитики)
